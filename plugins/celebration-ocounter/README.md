# Celebration O-Counter

Replaces Stash's default O-counter drops with a discreet celebration sparkles
icon. The plugin changes only the icon: incrementing, decrementing, resetting,
history, tooltips, and counter values continue to use Stash's standard behavior.

The icon inherits the active theme color. When Stash's SFW content mode is
enabled, Stash continues to show its standard thumbs-up icon.

## Installation

Add this package source in **Settings > Plugins > Available Plugins**:

```text
https://mrx-31415.github.io/stash-plugins/main/index.yml
```

Reload the available packages, then install **Celebration O-Counter**.

## Compatibility

This plugin uses Stash's UI plugin API and the named `SweatDrops` component
patch point. Plugin support is experimental, so a future Stash release may
require a compatibility update.

## Uninstallation

Uninstall the plugin from **Settings > Plugins** to restore the stock icon.
